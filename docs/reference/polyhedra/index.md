# Polyhedra


## Overview

|                | Tetrahedron    | Hexahedron     | Octahedron     | Dodecahedron   | Icosahedron    |
|----------------|----------------|----------------|----------------|----------------|----------------|
| Faces          | 4              | 8              | 6              | 20             | 12             |
| Vertices       | 4              | 6 (2 × 3)      | 8              | 12 (4 × 3)     | 20 (8 + 4 × 3) |






## Parameters

| Symbol         | Parameter      | Description    |
|----------------|----------------|----------------|
| a              | Edge length    |                |
| r              | Radius         |                |
| A0             | Face area      |                |
| A              | Surface area   |                |
| h              | Height         |                |
| V              | Volume         |                |
| l              |                | Edge to opposite edge distance   |
