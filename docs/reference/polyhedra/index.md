# Polyhedra


## Overview

|                | Tetrahedron    | Hexahedron     | Octahedron     | Dodecahedron   | Icosahedron    |
|----------------|----------------|----------------|----------------|----------------|----------------|
| Faces          | 4              | 8              | 6              | 20             | 12             |
| Vertices       | 4              | 6 (2 × 3)      | 8              | 12 (4 × 3)     | 20 (8 + 4 × 3) |






## Parameters

| Parameter      | Symbol         | Name      | Description    |
|----------------|----------------|----------------|
| a              | $a$ | Edge length    |                |
| r              | $r$               | Radius         |                |
| r_u            | $r_u$               |          |                |
| r_i            | $r_i$               |          |                |
| r_m            | $r_m$               |          |                |
| A0             | $A0$               | Face area      |                |
| A              | $A$               | Surface area   |                |
| h              | $h$               | Height         |                |
| V              | $V$               | Volume         |                |
| l              | $l$               |                | Edge to opposite edge distance   |
